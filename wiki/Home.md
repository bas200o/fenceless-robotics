### VM Warning
Using a virtual machine with the realsense camera requires a USB 3.1 datarate passthrough. The VM might impact performance or the guest might require extra software for graphics accelleration.  
Using a vm is possible although not recommended.   
If you do use a VM it's recommended that you use VMWare Workstation on Windows.

### ZED warning
ZED SDK requires CUDA this means an nvidia gpu.  
If you use a VM you need to use gpu passthrough.  
If you have a Intel or AMD GPU you need to add a Nvidia GPU
